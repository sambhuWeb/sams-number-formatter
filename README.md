Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install @rajsamb/sams-number-formatter`

## Usage

    var numFormatter = require('@rajsamb/sams-number-formatter');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`
